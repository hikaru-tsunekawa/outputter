# Outputter概要

## コンセプト
- 誰でも気軽にアイデアや知識をアウトプットすることができるアプリ

## 制作のきっかけ
- 情報過多な現代ではいくら調べてもインプットするだけで終わってしまうことが多い。
- インプットしたことを自分なりのアイデアに発展させたい
